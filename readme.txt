Desafio Rambo Webs. 

Tiempo limite 30 de junio o hasta que los 2 participantes terminen las 3 webs

Las reglas son las siguientes. Solo HTML y CSS, se puede utilizar cualquier ayuda de internet o chat GPT sin abusar. Esto es para practicar 
y divertirse. Importante, no desubicarse en los textos ya que esto les puede servir para porfolio. Aunque un poco de humor le va a jugar a favor.

La estructura puede no ser la clasica, de header con botones, pie de pagina con descripcion y creditos. Aunque se puede hacer de esta manera.


Tema 1. Tienda de naves alienigenas. 

Tiene que tener fotos de las naves, precio, en la moneda que quieran.
Se valora cualquier descripcion creativa de cada una de las naves.


Tema2. Galeria de paisajes del world of warcraft.
Puede contener imagenes tanto de paisajes como de los moustros que se encuentran en cada paisaje-mapa.
Se valora descripcion creativa.
